# 🛍️ Retail Inventory Management System

A simple and efficient **Retail Inventory Management System** built using **HTML, CSS, and Flask (Python)**. This project helps small businesses and retailers manage their stock, track products, and streamline basic inventory operations through a clean web interface.

---

## 📌 Overview

This project is a web-based application designed to:

* Manage product inventory
* Add, update, and delete items
* Track stock levels in real time
* Provide a simple UI for easy navigation

It is ideal for beginners learning full-stack development with **Flask** and frontend technologies.

---

## 🚀 Features

* ➕ Add new products with details (name, quantity, price)
* ✏️ Update existing inventory items
* ❌ Delete products from inventory
* 📦 View all available stock in a structured format
* 🔍 Simple and clean user interface
* ⚡ Fast and lightweight backend using Flask

## ⚙️ Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/retail-inventory.git
   cd retail-inventory
   ```

2. **Create a virtual environment (optional but recommended)**

   ```bash
   python -m venv venv
   source venv/bin/activate   # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**

   ```bash
   pip install flask
   ```

4. **Run the application**

   ```bash
   python app.py
   ```

5. **Open in browser**

   ```
   http://127.0.0.1:5000/
   ```
